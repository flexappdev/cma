# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Repo Is

Three-part learning system for **Claude Managed Agents** (Anthropic, public beta April 2026):

- `app/` — Next.js 15 study dashboard (port **24202**)
- `docs/claude-architect-certification.md` — 7-module certification guide with 25 assessment questions
- `.claude/skills/cma/SKILL.md` — installable `/cma` Claude Code skill

## App Commands

All commands run from `app/`:

```bash
npm run dev      # Start dev server at localhost:24202
npm run fresh    # Wipe .next cache and restart
npm run build    # Production build
npm run lint     # ESLint
npx tsc --noEmit # Type check
```

## App Architecture

**Provider tree** (layout.tsx): `SettingsProvider` → `ProgressProvider` → `ShellWrapper` → page

**Shell** (`ShellWrapper`): fixed `LeftSidebar` (collapsible, 180px/52px) + `main` content area + fixed `AppFooter`

**Two data sources:**
1. `src/lib/modules.ts` — static array of `MODULES` (7 modules, lessons, quiz questions, resources). This is the source of truth for the learning curriculum and is imported directly by components — no API call.
2. `app/courses.json` — 12 course objects read by `src/app/api/courses/route.ts`, returned with filtering/sorting/pagination. Used by `CourseGallery` via `/api/courses`.

**Progress tracking** (`src/lib/progress.tsx`): `ProgressProvider` stores completed lesson keys in localStorage under `cma-lesson-progress`. Keys are `moduleId::lessonId` (generated by `lessonKey()`). Components call `useProgress()` for `{ completed, toggle, isCompleted, getModuleProgress }`.

**Settings** (`src/lib/settings.tsx`): `SettingsProvider` stores `{ accentColor, navCollapsed }` in localStorage under `cma-settings`. Applies CSS variables `--app-accent`, `--app-accent-dark`, `--app-accent-light` to the document root.

**Pages:**
- `/` — ModuleGrid (reads MODULES)
- `/modules` — same
- `/modules/[id]` — LessonList for a specific module
- `/quiz` — QuizCard pulling from QUIZ_QUESTIONS in modules.ts
- `/progress` — per-module completion rings
- `/resources` — RESOURCES array from modules.ts
- `/settings` — accent colour picker, sidebar toggle

## Adding Content

**New lesson:** add to the relevant module's `lessons` array in `src/lib/modules.ts`. Use `id` as a URL-slug-safe string; `duration` as `"Xmin"`.

**New quiz question:** add to `QUIZ_QUESTIONS` in `src/lib/modules.ts` with `module: 1–7`.

**New course card:** add to `app/courses.json`. Valid `level`: `beginner|intermediate|advanced`. Valid `status`: `published|draft|coming-soon`.

## Key Conventions

- **Path alias:** `@/*` → `src/*`
- **Tailwind target:** dark theme, zinc/slate palette, orange primary (`--app-accent` CSS var)
- **No database** — all data is either static (modules.ts) or file-based (courses.json)
- **Set iteration** must use `Array.from()` not spread (`[...set]`) due to TS target ES2017
- **"use client"** required on any component using hooks, context, or browser APIs
